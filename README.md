# cl-webapp-seed
A simple web application boilerplate using Common Lisp 

## Dependencies
Hunchentoot
Parenscript
Smackjack
cl-who

To run the demo locally, clone the repo and load "local-server.lisp" in you LISP environment.

Eg: In SBCL
```
sbcl --load "local-server.lisp"
```
